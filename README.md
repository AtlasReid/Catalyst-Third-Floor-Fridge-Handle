# Catalyst Third-Floor Fridge Handle

This repository contains the design files for a 3D-printed fridge handle created for the third floor of the Catalyst building at Newcastle Helix.

The handle is intended to make the fridge door easier to open for people working on the floor.

## About the Design

This is the third version of the handle.

Compared with the earlier versions, this design:

- fits the current fridges more securely;
- is easier to grip and use;
- has a simpler shape that should be easier to clean;
- is designed to be reprinted and replaced if it becomes damaged.

## Fridge Compatibility

There are two different handle versions because the fridges use different mounting-slot profiles.

- **Left-hand fridge:** NICA/NICRE
- **Right-hand fridge:** NIHRIO

The adapter profiles are different and are designed to fit their respective fridge slots. The two handles are therefore **not interchangeable**.

Please check that you are using the correct file before printing.

## Design Files

The design was created using **SolidWorks 2026**.

For slicing and 3D printing, it is recommended that you use the supplied **STEP file** rather than the native SolidWorks file. This should provide better compatibility with different slicing software and avoid issues caused by SolidWorks version differences.

## Reprinting the Handle

These files have been uploaded so that replacement handles can be produced in the future if the existing handles are lost or damaged.

The nearest suitable fabrication facility is:

**Open Lab Maker Space**  
First Floor, Urban Sciences Building  
Newcastle Helix

Before printing, confirm:

1. which fridge the handle is for;
2. that the correct adapter profile has been selected;
3. that the model dimensions have not been altered during import;
4. that the chosen material and print settings are suitable for regular handling and cleaning.

## Installation

Slide the correct adapter into the existing mounting slot on the fridge door.

Do not force the handle into place. If it does not fit easily, check that the correct version has been printed.

## Maintenance

Clean the handle using a damp cloth and an appropriate general-purpose cleaning product.

Avoid using excessive heat or harsh chemicals that may soften, deform, or damage the printed material.

Inspect the handle periodically for cracks, looseness, or other signs of damage. Replace it if its condition could affect safe use.
